# Use Case: View Assignment


**Primary Actor:** Student  
**Secondary Actor:** Instructor

**Goal:** Allow the student to view assignments created by the instructor.

**Preconditions:**  
- At least one assignment has been created by the instructor.

**Postconditions:**  
- The details of the selected assignment are displayed to the student.

**Main Scenario:**  
1. The instructor has created an assignment.  
2. The student selects the option to view assignments.  
3. The system displays the details of the available assignments.

| Learning Management System: View Assignment | |
|--------------------------------------------|---|
| Actors      | Student, Instructor |
| Description | The student can view information about assignments created by the instructor. |
| Data       | Assignment details (title, description, due date) |
| Stimulus   | Student requests to view assignments |
| Response   | The system displays available assignments |
| Comments   | Shows which instructor issued the assignment |
