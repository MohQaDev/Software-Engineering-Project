# Use Case: View Performance Report

**Primary Actor:** Student  
**Secondary Actor:** Instructor

**Goal:** Allow the student to view their performance report generated from grades managed by the instructor.

**Preconditions:**  
- The instructor has entered or managed grades for the student.  

**Postconditions:**  
- The student’s performance report is displayed.  

**Main Scenario:**  
1. The instructor manages grades for assignments.  
2. The system generates the performance report based on the entered grades.  
3. The student selects the option to view their performance report.  
4. The system displays the student’s performance report.

| Learning Management System: View Performance Report | |
|-----------------------------------------------------|---|
| Actors      | Student, Instructor |
| Description | The student can view their performance report generated from grades managed by the instructor. |
| Data       | Grades, assignment scores, performance metrics |
| Stimulus   | Student requests to view performance report |
| Response   | The system displays the student’s performance report |
| Comments   | The report reflects grades entered by the instructor |
